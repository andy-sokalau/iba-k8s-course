Practical tasks:

Note: for the following tasks save used commands and/or results to a file (you can do screenshots) and send to the tutor for review. Provide a Docker ID.

1) Create an account on Docker Hub
2) Log in to the hub on your Docker host
3) Create docker-course repository in your docker hub account
4) Tag the image from the section "Build Docker image" with your own Docker Hub repository to push it
5) Push your image to the Docker hub repository (Once the push is finished you will be able to see the image in your Docker Hub page.)
6) Pull the image from the repository
